# Support-ticket-api-simulation

## Overview
This project simulates a support ticket system using REST APIs.

It demonstrates:
- Retrieving data (GET)
- Creating tickets (POST)
- Updating tickets (PUT)
- Deleting tickets (DELETE)

Built and tested using Postman.
